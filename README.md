# ⚖️ Speeding Ticket Cost Calculator (2026 Edition)

A high-performance, mobile-optimized suite of calculators designed to estimate the **total financial impact** of speeding violations across all 51 US states. 

Live Demo: [https://nafiyial-jpg.github.io](https://nafiyial-jpg.github.io)

## 🚀 Key Features

- **51-State Support:** Customized calculation logic for every US state including DC.
- **2026 Insurance Surcharge Logic:** Estimates 3-year premium hikes based on current actuarial trends.
- **Mobile-First Design:** Featuring numeric keypad triggers and smooth scrolling for a seamless mobile experience.
- **SEO Optimized:** Structured data (JSON-LD) and a full `sitemap.xml` for maximum search engine visibility.
- **Privacy Centric:** All calculations are performed client-side. No personal driving data is ever stored or transmitted.

## 🛠️ Project Structure

- `index.html`: The main portal featuring a state-by-state navigation grid.
- `privacy.html`: Transparent privacy policy focused on data protection.
- `sitemap.xml`: Full site index for Google Search Console indexing.

## 📈 SEO & Performance

This project is built with **Inter (Google Fonts)** for a modern typography feel and uses a high-contrast UI (Slate & Blue) to drive user engagement and conversion for insurance comparison tools.

## ⚖️ Disclaimer

The estimates provided by this tool are for informational purposes only. Actual fines and insurance surcharges may vary based on individual driving records and specific court rulings.
